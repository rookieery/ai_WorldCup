# Skill: championship_predict_2018_backtest — 2018世界杯冠亚军预测回测分析

## 一、元信息

- **版本**: 1.0
- **适用赛制**: 2018年FIFA世界杯（32队 / 8组 / 前2名出线 / 16强淘汰赛）
- **核心目的**: 回测2026冠亚军预测策略在2018赛事上的表现，与2022回测交叉验证，进一步校准因子权重
- **回测方法**: 10轮 × 1000次蒙特卡洛模拟（共10,000次独立推演）
- **输入时点**: 2018年6月14日开赛前的所有可用数据
- **核心结论**:
  - ✅ 实际冠军（法国）在预测中排名第4（10.2%），10轮均稳定落入TOP5
  - ❌ 实际决赛对阵（法国 vs 克罗地亚）**未进入TOP20预测**（概率仅~0.4%）——这是与2022回测最显著的差异
  - ❌ 实际亚军（克罗地亚 FIFA#20）在预测中排名第11（2.0%），**未进入TOP5**
  - ⚠️ 上半区为明确的"死亡半区"（巴西#2/法国#7/比利时#3/葡萄牙#4），模型正确识别
  - 🔑 东道主效应（俄罗斯 FIFA#70→8强）和卫冕冠军魔咒（德国 FIFA#1→小组垫底）是2018独有的关键因子
- **与2022对比**：2022回测中冠军和决赛对阵均落入TOP3；2018回测中冠军落入TOP5但决赛对阵**严重偏离**——两届回测共同校准可显著提升2026预测鲁棒性

## 四、蒙特卡洛回测设计

模拟参数与2022回测完全一致（10轮×1000次），确保两届回测可比。

### 4.1 小组赛预测模型

```
同2022回测模型:
  sigmoid(0.065 * diff) + 种子队修正 + 东道主修正

新增（2018特有）:
  IF 球队 = 卫冕冠军: base_prob += 0.03  (微弱加成)
  IF 球队 = 东道主: base_prob += 0.10  (东道主优势)
```

### 4.2 淘汰赛预测模型

```
同2022回测模型:
  sigmoid(0.065 * diff) + 策略六Grit修正(排名差<12) + 点球逻辑

半区路径修正:
  IF 半区TOP4平均FIFA排名差距 > 3:
    弱半区决赛代表获得"路径优势溢价" (+5% 夺冠概率)
  IF 半区标记为DEATH_HALF:
    该半区SF代表队决赛胜率下调 5%
```

## 七、策略校准与优化建议（反哺2026模型）

### 7.1 东道主效应修正 ⭐2018独有发现

**证据**：俄罗斯(FIFA#70)小组出线 → R16击败西班牙(身价第1) → QF仅点球负克罗地亚。东道主效应远超模型预期。

**新增修正**：

```
东道主效应修正:
  IF 球队 = 东道主:
    小组赛: base_prob += 0.12  (原0.05, 提升至0.12)
    R16: base_prob += 0.10
    QF: base_prob += 0.05
    SF+: base_prob += 0.00  (后期轮次东道主效应衰减)

  IF 东道主 FIFA排名 > 50:
    额外加成: 小组出线概率 += 20%
    (反映"举国动员"效应——球迷支持、裁判偏护、赛前集训优势)
```

**2026影响**：2026为美国/加拿大/墨西哥联合举办，三国均为参赛队。其中美国和墨西哥实力较强，东道主效应可能显著影响淘汰赛对阵。

### 7.2 卫冕冠军魔咒修正 ⭐2018独有发现

**证据**：连续5届卫冕冠军小组出局（2002法→2018德）。这已不再是"巧合"而是系统性规律。

**新增修正**：

```
卫冕冠军魔咒修正:
  IF 球队 = 上一届世界杯冠军:
    小组赛: base_prob *= 0.75  (下调25%)
    IF 卫冕冠军 FIFA排名前3:
      额外惩罚: base_prob *= 0.90  (排名越高，魔咒越重)
    IF 卫冕冠军 核心阵容老化(平均年龄>28):
      额外惩罚: base_prob *= 0.85

  2026目标: 阿根廷(2022冠军) 需应用此修正
```

### 7.3 死亡半区存活者溢价（新增发现）

**证据**：法国从死亡半区（含巴西/比利时/葡萄牙）杀出后，在决赛中4-2轻取来自弱半区的克罗地亚。死亡半区的存活者具有显著的心理和竞技优势。

**新增修正**：

```
死亡半区存活者溢价:
  IF 决赛一方来自DEATH_HALF, 另一方来自弱半区:
    DEATH_HALF存活者 决赛胜率 += 0.08
    (反映"淘汰赛高强度历练"的竞技状态优势)

  IF DEATH_HALF存活者在晋级过程中击败了 ≥ 2支 TOP10 球队:
    额外加成: 决赛胜率 += 0.05
```

### 7.4 累积疲劳因子（克罗地亚模式）

**证据**：克罗地亚在2018淘汰赛中连续3场进入加时赛/点球大战（vs丹麦、vs俄罗斯、vs英格兰）。这是历史上首次有球队在单届世界杯连续3场淘汰赛打加时。虽然克罗地亚最终进入决赛，但在决赛中明显体能不足（0-2落后、4-2输球）。

**新增修正**：

```
累积疲劳因子:
  FOR each 球队:
    et_count = 淘汰赛进入加时赛的次数
    IF et_count >= 2:
      后续比赛 base_prob *= (1 - 0.05 * et_count)
      (每多打一场加时, 胜率下降5%)
    IF et_count >= 3:
      额外惩罚: base_prob *= 0.85
      (3场加时后, 体能严重透支)
```

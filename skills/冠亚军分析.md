# Skill: championship_predict — 2026世界杯冠亚军预测分析

## 一、元信息

- **版本**: 2.2（2018+2022双重复测校准版）
- **适用赛制**: 2026年FIFA世界杯（48队 / 12组 / 前2名 + 8个最佳第3名出线 / 32强淘汰赛）
- **分析粒度**: 全赛事冠亚军预测（蒙特卡洛模拟思想）

- **回测基准**: 已通过2018+2022双重复测验证（各10轮×1000次蒙特卡洛），详见 `冠亚军分析-2018回测.md` 和 `冠亚军分析-2022回测.md`

## 三、七大核心预测策略详解

### 策略优先级矩阵

| 冲突场景 | 优先级裁决 | 说明 |
|----------|----------|------|
| 策略四 vs 策略一 | **策略四优先** | 特赦覆盖惩罚。FIFA排名前10的第三名恢复全部实力权重 |
| 策略三 vs 所有概率策略 | **策略三优先（硬约束）** | 分区碰撞检测是"绝对禁止"，不是概率调整 |
| 策略七 → 策略二 | **策略七修正策略二的输入** | 时间序列博弈修正小组赛末轮结果，改变32强初始名单 |
| 策略六 vs 策略二 | **策略六叠加修正** | 杯赛韧性在淘汰赛阶段叠加修正纸面实力基线 |
| 策略八 vs 策略二 | **策略八修正策略二的基线** | 黄金一代衰退下调"纸面强队"的初始实力评估 |
| 策略九 vs 策略一 | **策略九对冲策略一** | 防守型黑马即使排名低，也获得韧性加成（与第三名惩罚逻辑独立） |

### 策略一：第三名逆袭惩罚（Underdog Pruning）

**策略背景**：扩军后引入8个成绩最好的小组第三。在历史杯赛中（1996/2016/2020欧洲杯），小组第三通过单场爆冷杀入决赛的概率极低。战线拉长至8场（夺冠所需）后，板凳深度不足的第三名很难走远。

**执行指令**：

```
FOR each 球队 X in 32强名单:
    IF X 是"最佳第三名"出线:
        IF X 满足策略四的特赦条件:  → 跳过惩罚，转策略四处理
        ELSE:
            X.underdog = True
            对阵小组第1名时: X 胜率 *= 0.70  (降低 30%)
            对阵小组第2名(FIFA排名 ≤ 15)时: X 胜率 *= 0.75
            对阵小组第2名(FIFA排名 > 15)时: X 胜率 *= 0.85
    IF X 是 underdog 且在 QF 及之后仍存活:
        每存活一轮: 胜率惩罚回收 10%
        (QF: -20%, SF: -10%, F: 恢复全部权重)
```

**数据降级**：缺少FIFA精确排名时，按小组赛表现（积分/净胜球/进球数）近似判断是否为"传统弱队"。

### 策略二：纸面实力导向（Seed Squad Dominance）

**策略背景**：小组赛有4支球队，前两名直接出线，爆冷容错率较高。传统豪门即使偶尔打盹，也大概率凭借硬实力锁前两名。

**执行指令**：

```
1. 提取硬指标:
   - FIFA世界排名 (primary)
   - 全队市场身价 (secondary, 缺失时仅用排名)

2. 实力差计算:
   diff = rank_away - rank_home  (正值 = 主队排名更靠前)
   k = 0.065  (sigmoid 曲率系数)
   base_win_prob = 1 / (1 + exp(-k * diff))

3. 小组赛出线概率:
   IF 排名差 > 20 位: 前2名球队晋级概率 ≥ 75%
   IF 排名差 10-20 位: 晋级概率 55%-75%
   IF 排名差 < 10 位: 激烈竞争组，需叠加策略七的控分分析

4. 该策略作为初始化底色，生成32强初始名单
```

**数据降级**：缺少市场身价时，仅用FIFA排名作为实力指标，sigmoid系数 k 不变。

### 策略三：分区碰撞检测（Bracket Collision Detection）⭐核心

**策略背景**：2026赛制下，所有12个小组的第1名和第2名均被分入不同半区（见2.3节半区分布表）。因此，**同组的1号和2号球队只有决赛才能相遇，不可能在更早的轮次产生内战**。碰撞检测的核心不是"同组内战"，而是识别**同一半区内不同小组强队之间的"死亡路径"**。

**执行指令**：

```
STEP 1: 读取2.1/2.2节对阵表，确认每支球队的半区归属

STEP 2: 对每个半区分别执行路径推演:

  FOR each 半区 (SF_01, SF_02):
    收集该半区所有可能的参赛球队（含第三名候选）
    评估每支球队的"路径难度":
      path_difficulty = SUM(潜在对手的FIFA排名倒数) / 对手数量
    标记"死亡路径":
      IF 半区内存在 ≥ 3 支 FIFA排名前8的球队:
        该半区标记为 "DEATH_HALF"
        死亡半区的半决赛胜者晋级概率需下调 5-10%
        (因为晋级过程中消耗更大)

STEP 3: 硬约束检查（不可违反）:

  ① 同组1号和2号: 允许同时出现在决赛预测中（分属不同半区）
  ② 同一R32比赛的双方: 只有1支能晋级R16
  ③ 第三名球队: 不可与同组的1号/2号在R32相遇（约束表已保证）

STEP 4: 半区强度对比:

  upper_strength = 上半区 TOP4 球队的平均FIFA排名
  lower_strength = 下半区 TOP4 球队的平均FIFA排名
  strength_gap = |upper_strength - lower_strength|
  IF strength_gap > 5:
    弱半区的决赛代表队获得 "路径优势溢价" (+5% 夺冠概率)
```

**数据降级**：缺少精确排名时，用常识性的"传统豪门 / 准一流 / 二流"三级分类替代。

### 策略四：超级"死亡之组"第三名特赦（Powerhouse Amnesty）

**策略背景**：策略一降低了第三名的权重，但若发生"法国、意大利、塞内加尔"同在一组的极端情况，可能有世界顶级豪门不幸沦为第三名出线。这类球队具备直接夺冠的硬实力。

**执行指令**：

```
FOR each 第三名出线球队 X:
    IF X.FIFA排名 ≤ 10 (即世界前10):
        X.amnesty = True
        → 抹除 underdog 标签
        → 恢复 100% 真实实力权重
        → 额外微调激励: 胜率 += 0.05
          (反映"背水一战"的斗志加成)
    ELIF X.全队市场身价 位列所有48支球队 Top 12:
        X.amnesty = True
        → 同上处理
    ELSE:
        → 维持策略一的惩罚逻辑
```

| 判定条件 | 特赦结果 | 与策略一的交互 |
|----------|---------|-------------|
| FIFA排名 ≤ 10 | 完全特赦 + 激励 | 策略四优先，覆盖策略一惩罚 |
| 身价 Top 12 | 完全特赦 + 激励 | 策略四优先，覆盖策略一惩罚 |
| 不满足以上条件 | 维持策略一惩罚 | 策略一正常执行 |

**数据降级**：缺少身价数据时，仅用FIFA排名判断（排名 ≤ 10 即特赦）。无排名数据的球队默认不触发特赦。

### 策略五：第三名落位蒙特卡洛矩阵（Dynamic 3rd-Placers Matrix）

**策略背景**：哪8个小组第三能出线是动态的。FIFA官方根据这8个第三名的不同组合，按2.4节约束表动态映射到R32的具体比赛。不同组合会导致对阵格局剧烈变化。

**执行指令**：

```
STEP 1: 估算各小组第三名的出线概率
  FOR each 小组 G:
    该组第3名的出线概率 ≈ f(该组实力差距, 该组前2名的确定性)
    IF 该组有2支以上FIFA排名前20的球队:
      第3名出线概率下降 20-30% (因为第3名本身可能很弱)
    IF 该组实力接近:
      第3名出线概率上升 (因为第3名可能拿到较高积分)

STEP 2: 筛选最可能的8个第三名
  按出线概率排序，取前8个
  第三名出线的积分门槛估算: ≥ 4分 (1胜1平) 为安全线
  3分 (1胜2负 或 3平) 为临界线，需要比较净胜球

STEP 3: 按约束表落位
  将筛选出的8个第三名，按2.4节约束表映射到对应的R32比赛
  注意: 不同的组合 → 不同的映射结果
  重点评估: 哪些种子队(E1/I1/A1/L1/G1/D1/B1/K1)可能遭遇强敌第三名

STEP 4: 敏感度分析
  IF 某强队(E1/A1/B1等)的首轮对手可能是特赦级第三名:
    该强队的晋级路径难度上调 15-20%
```

**数据降级**：LLM无法执行1000次蒙特卡洛模拟，应采用"最可能出线的8个第三名"的定性推理，聚焦于识别哪些种子队可能遭遇强敌第三名。

### 策略六：杯赛防御/点球加权修正（Tournament Grit Factor）

**策略背景**：淘汰赛战线由4轮拉长至5轮（R32→R16→QF→SF→F）。纸面实力强的攻势型球队，极易被擅长防守反击、拖点球大战的"杯赛专家型"球队阻击（如历史上的克罗地亚、摩洛哥）。

**执行指令**：

```
1. 为所有球队引入"杯赛韧性系数"(Grit Factor)，提取三个指标:
   - 近三届大赛场均失球数 (越低越好)
   - 淘汰赛点球大战胜率 (Penalty Shootout Win Rate)
   - 零封率 (Clean Sheet Rate)

2. 触发条件（2022回测校准，v2.1更新）:
   IF 两队纸面实力差距 < 20 位(基于FIFA排名):        ← 原12放宽到20
     grit_diff = grit_A - grit_B
     grit_bonus = grit_diff * 0.35                    ← 原0.25提升至0.35
   OR IF 较弱队 Grit Factor > 0.65:                   ← 杯赛专家无条件触发
     同上计算

3. 轮次加成:
     IF 当前轮次 ∈ {QF, SF, F}:
       grit_bonus *= 1.5  (后期比赛韧性更重要)        ← 原1.3提升至1.5

4. 效果:
     胜率调整: p += grit_bonus * 0.08
     点球大战修正: 点球胜率 = 0.5 + grit_diff * 0.15  ← 原0.10提升至0.15
     胜率夹值: [0.05, 0.95]  (确保不出现0或1的极端值)

> ⚠️ v2.1校准说明：2022回测发现，原版策略六在4场关键爆冷中3场未触发
> （摩洛哥vs西班牙排名差15、摩洛哥vs葡萄牙排名差13、克罗地亚vs日本排名差12）。
> 放宽门槛后可覆盖所有关键场景，预期黑马识别率提升30%。
```

**数据降级**：缺少精确统计时，使用以下简化评估：

| 球队类型 | Grit Factor 近似值 | 代表球队 |
|---------|-------------------|---------|
| 杯赛专家（传统防守强队） | 0.7-0.8 | 克罗地亚、摩洛哥、乌拉圭 |
| 攻守均衡的传统豪门 | 0.5-0.6 | 法国、德国、阿根廷 |
| 攻势足球但杯赛经验不足 | 0.3-0.4 | 新兴球队、首次参赛球队 |

### 策略七：时间序列博弈与非对称信息控分（Chronological Game Theory）

**策略背景**：12个小组的第三轮比赛按2.5节时间线顺序开球。越晚开赛的球队，掌握的淘汰赛落位情报越透明。如果上半区的霸主已锁定席位，随后出战的球队在明确获胜将与之提前遭遇时，有极强动机主动放弃胜利，以小组第二身份"潜逃"至另一半区。

**执行指令**：

```
1. 信息透明度计算:
   group_index = 该组在时间线中的位置 (B=0, C=1, A=2, ..., J=11)
   info_clarity = group_index / 11.0  (范围 0.0~1.0)

2. 控分动机检测:
   FOR each 第三轮比赛中的球队 X:
     预测X获胜后的淘汰赛落位路径 → 评估路径难度 path_A
     预测X平局后的淘汰赛落位路径 → 评估路径难度 path_B
     预测X失利后的淘汰赛落位路径 → 评估路径难度 path_C
     IF (path_B < path_A) 或 (path_C < path_A):
       X 具有"控分动机"

3. 信息迷雾法则:
   IF info_clarity ≤ 0.5 (前6组: B/C/A/E/F/D):
     → 信息迷雾中，禁止过度控分推演
     → 锁定为正常竞技结果
   IF info_clarity > 0.5 (后6组: I/H/G/L/K/J):
     → 信息透明，允许全景控分推演

4. 控分安全着陆:
   控分是危险的走钢丝行为，优先推演以下安全结果:
   - 平局 (最安全的控分结果)
   - 微弱劣势输球 (1球差)
   避免推演"大比分输球"这种不现实的控分场景

5. 默契球检测:
   IF 对阵双方都有控分动机 (双方都不想赢):
     → 常规实力指标失效
     → 大幅拉升平局概率 (最高至 0.45)
     → 重点输出 0-0 或 1-1 的小比分平局
```

**数据降级**：缺少精确的开球时间时，默认按"先抽签的组先赛"的假设处理。缺少历史控分案例时，仅对后4组（G/L/K/J）开放控分推演。

### 策略八：黄金一代衰退检测（Generational Decline Detection）⭐2022回测新增

**策略背景**：2022回测发现，FIFA排名无法区分"真实强队"和"纸老虎"。比利时(FIFA#2)、丹麦(FIFA#10)、德国(FIFA#11)均在小组赛出局，核心原因是黄金一代末期阵容老化或内部分裂。该策略用于在赛前识别这类"衰退中的纸面强队"，避免过度高估。

**执行指令**：

```
STEP 1: 检测"黄金一代衰退"信号
  FOR each 参赛球队 X:
    signal_count = 0
    IF X 核心阵容(首发11人)平均年龄 > 29.5: signal_count += 1
    IF X 核心阵容 ≥ 4人来自同一黄金代且 age > 30: signal_count += 1
    IF X 近1年正式比赛胜率 < 50% 且 FIFA排名前15: signal_count += 1
    IF X 近期有公开的内部矛盾/换帅/战术动荡报道: signal_count += 1

STEP 2: 分级修正
    IF signal_count ≥ 3: X.decline = "severe"
      → 小组出线概率 *= 0.70
      → 淘汰赛 base_prob *= 0.75
    ELIF signal_count == 2: X.decline = "moderate"
      → 小组出线概率 *= 0.85
      → 淘汰赛 base_prob *= 0.85
    ELIF signal_count ≤ 1: 正常处理

STEP 3: 特殊场景
    IF X 是卫冕冠军 且 signal_count ≥ 1:
      → 额外下调 5%（冠军魔咒叠加衰退）
```

**数据降级**：缺少精确年龄和胜率数据时，使用以下定性信号替代：
- 核心球员是否已连续参加3届以上大赛
- 近12个月是否有公开的战术/阵容动荡
- 球队近2届大赛成绩是否呈下降趋势

### 策略九：防守型黑马识别（Defensive Dark Horse Detection）⭐2022回测新增

**策略背景**：2022回测中，摩洛哥(FIFA#22)杀入4强、克罗地亚(FIFA#12)获得季军，两队的共同特征是极强的防守韧性和点球大战能力。这类"低排名高韧性"球队在传统实力模型中被严重低估。该策略用于在赛前识别潜在的防守型黑马。

**执行指令**：

```
STEP 1: 识别防守型黑马候选
  FOR each FIFA排名 > 15 的球队 X:
    dark_horse_score = 0
    IF 近10场正式比赛场均失球 < 0.8: dark_horse_score += 2
    IF 近10场正式比赛零封率 > 40%: dark_horse_score += 2
    IF Grit Factor > 0.65: dark_horse_score += 2
    IF 所在大洲有"旅途/气候/文化"优势: dark_horse_score += 1

STEP 2: 修正
    IF dark_horse_score ≥ 4: X.dark_horse = True
      对阵排名高于自己的球队时:
        X 胜率 += 0.08
        平局概率 += 0.05
        进入点球大战的概率 += 0.05
    IF dark_horse_score ≥ 6: X.dark_horse_elite = True
      额外加成:
        X 淘汰赛每存活一轮, 胜率 += 0.03（黑马士气递增）

STEP 3: 输出
    列出所有 dark_horse 球队及其预测最远轮次
    特别标记可能在 R32 阻击种子队的黑马
```

**数据降级**：缺少精确统计时，使用以下简化规则：
- 该队是否以防守反击为主要战术（查看近期比赛阵型和控球率）
- 该队门将是否为世界级（大赛经验丰富的门将是点球大战的关键）
- 该队是否有"大赛超常发挥"的历史传统

## 四、推理步骤链（Reasoning Chain）

### STEP 1: 小组赛模拟 → 32强名单生成

**目标**：基于策略二（纸面实力）和策略七（时间序列控分），预测12个小组的最终排名，生成24支出线队（前2名）的初始名单。

**执行指令**：
1. 按2.5节时间线顺序，依次推演12个小组第三轮结果
2. 前两轮采用策略二的实力基线
3. 第三轮叠加策略七的控分分析（info_clarity > 0.5 时触发）
4. 产出：12个第1名 + 12个第2名 = 24支出线队

**输出**：
```
{
  "group_rankings": {
    "A": {"1st": "...", "2nd": "...", "3rd": "...", "4th": "..."},
    ...
  },
  "confirmed_32": ["A1", "A2", "B1", "B2", ...],  // 24支确定球队
  "third_place_pool": ["A3rd", "B3rd", ...]  // 12个第三名候选
}
```

---

### STEP 2: 第三名筛选与动态落位 ⭐核心步骤

**目标**：从12个第三名中筛选出8个最佳第三名，按2.4节约束表动态分配到R32比赛。

**执行指令**：
1. 按"积分 → 净胜球 → 进球数"排序12个第三名
2. 取前8名出线
3. 检查是否触发策略四（特赦条款）
4. 按2.4节约束表映射到8场涉及第三名的R32比赛
5. 评估每个种子队可能遭遇的第三名强度

**输出**：
```
{
  "qualified_thirds": ["C3rd", "F3rd", ...],  // 8个出线的第三名
  "third_placement": {
    "R32_03": "C3rd",  // E1 的对手
    "R32_06": "F3rd",  // I1 的对手
    ...
  },
  "amnesty_triggered": ["I3rd"]  // 触发特赦的第三名
}
```

---

### STEP 3: 半区碰撞检测与路径构建

**目标**：基于2.1/2.2节对阵表，对上半区和下半区分别构建完整的晋级路径树，标记"死亡路径"。

**执行指令**：
1. 将32支球队按对阵表填入R32的16场比赛
2. 逐轮推演R32→R16→QF→SF，每场应用策略一/四/六
3. 标记每个半区的"死亡路径"（≥3支排名前8球队聚集的路径段）
4. 产出上半区决赛代表和下半区决赛代表

**输出**：
```
{
  "upper_half_finalist": "...",  // 上半区进入决赛的球队
  "lower_half_finalist": "...",  // 下半区进入决赛的球队
  "death_paths": ["SF_01 路径: 包含 FRA/GER/ESP"],  // 死亡路径标记
  "path_difficulty": { "upper": 7.2, "lower": 5.1 }  // 半区难度评分
}
```

---

### STEP 4: 淘汰赛轮次推演

**目标**：对每个轮次（R32→R16→QF→SF）逐场推演，应用策略一/四/六的权重修正。

**执行指令**：
1. 每场比赛先算策略二的实力基线 (sigmoid 概率)
2. 叠加策略一/四的第三名惩罚/特赦
3. 叠加策略六的杯赛韧性修正（实力差 < 12 时触发）
4. 后期轮次（QF/SF）叠加 grit_bonus *= 1.3
5. 胜率夹值 [0.05, 0.95]

---

### STEP 5: 蒙特卡洛汇总与 TOP20 输出

**目标**：汇总多次推演结果，输出TOP20决赛场景的概率分布。

**执行指令**：
1. 重复 STEP 1-4 共 **{simulation_count}** 次（用户指定的蒙特卡洛模拟次数，不可更改）
2. 统计每种决赛组合的出现频次
3. 取 TOP20 高频组合
4. 对每个组合附上入选原因（引用具体策略）

**输出格式**：见第五节输出规范。

## 五、输出规范

### 5.1 TOP 20 决赛场景

对每个决赛场景，输出以下字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| 决赛对阵 | 两支球队的名称 | 法国 vs 阿根廷 |
| 出现概率 | 估算概率（百分比） | 12.3% |
| 入选原因 | 引用具体策略的推理链 | "法国通过下半区(SF_02)晋级，路径难度较低；阿根廷通过上半区(SF_01)晋级，但需在QF_01穿越死亡路径" |
| 半区归属 | 标明各队来自哪个半区 | 法国=下半区, 阿根廷=上半区 |

### 5.2 核心发现

必须包含以下分析：

1. **最可能冠军**：综合概率最高的球队及其晋级路径分析
2. **最大黑马**：非种子队中走最远的球队及原因
3. **死亡之组影响**：哪个小组对淘汰赛格局影响最大
4. **半区强度对比**：上下半区的整体实力差距及其对决赛的影响
5. **第三名变量**：哪些第三名出线可能颠覆对阵格局

### 5.3 数据约束

- 所有概率总和应 ≈ 100%（允许±5%误差）
- 同一小组的1号和2号可以同时出现在决赛中（分属不同半区）
- 不允许出现违反对阵表的晋级路径
- 每个场景必须引用至少一条策略作为依据

### 6.3 2026新增维度

1. **赛程密度**：冠军需打8场（3场小组赛 + 5场淘汰赛），比2022多1场R32，阵容深度不足的球队在第6-7场会显著下滑
2. **第三名不确定性**：首次引入8个最佳第三名，对阵表的第三名分配组合多达数十种，是最大的不确定变量
3. **北美气候/时区**：比赛横跨美国/加拿大/墨西哥三个国家、多个时区，长途旅行和气候适应是隐形因素

### 6.4 使用说明

- **独立使用**：将本文件完整输入大模型，配合具体的48支球队分组数据和FIFA排名，即可获得冠亚军预测分析
- **项目内调用**：通过API接口自动注入本文件作为系统提示的一部分
- **推荐搭配**：先使用 group_stage_predict 完成小组赛预测，再用本文件进行冠亚军推演，可获得更准确的结果

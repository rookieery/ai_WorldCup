### 1.1 整体命中率

| 指标 | 结果 |

| 淘汰赛总场次 | 16 |
| **90分钟方向命中** | **10/16 (62.5%)** |
| **晋级队伍命中** | **12/16 (75.0%)** |

> **关键说明**：2018淘汰赛是**平局密集型**赛事——16场中5场90分钟平局(31.25%)、4场点球大战。其中3场平局涉及东道主俄罗斯(淘汰西班牙、逼平克罗地亚)。本届策略的核心挑战是"近均势+东道主/点球专家"场景的平局预测——5场平局中策略捕获2场(哥伦比亚vs英格兰、克罗地亚vs英格兰)，3场rank悬殊的东道主爆冷(西班牙vs俄罗斯、俄罗斯vs克罗地亚)未能翻转方向但预警触发。**2018验证了2022提出的"连续加时体能考"——克罗地亚连续3场加时/点球，决赛体能崩盘0-4落后**。
